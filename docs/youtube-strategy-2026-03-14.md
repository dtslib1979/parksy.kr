## 1. 구독자 500명 글로벌 평균

| 지표 | 평균 |
|------|------|
| 영상 개수 | 50~100개 |
| 기간 | 6개월~1년 |
| 업로드 주기 | 주 1~2회 |

**결론:** 평균은 의미 없음. 콘텐츠 유형이 속도를 결정함.

| 유형 | 속도 |
|------|------|
| 검색형 (튜토리얼, 방법론) | 빠름 — 구글/유튜브 검색에 걸림 |
| 브이로그/일상 | 느림 — 팬덤 먼저 있어야 됨 |
| 쇼츠 | 조회수는 많은데 구독 전환율 낮음 |

---

## 2. 구독자 vs 돈 — 분리해서 생각

구독자와 수익은 별개.

| 경로 | 구독자 필요? | 핵심 지표 |
|------|------------|-----------|
| 유튜브 광고 (AdSense) | 1,000명 + 4,000시간 | 조회수, CPM |
| 쇼츠 보너스 | 500명 | 조회수 |
| 스폰서십 | 거의 무관 | 타겟 명확성 |
| 제휴 마케팅 | 무관 | 클릭률 |
| 자체 상품 | 무관 | 팬 충성도 |

**핵심:** 구독자 1만 + 타겟 명확 > 구독자 100만 + 타겟 불명확.
교육 카테고리 CPM이 높기 때문에 타겟이 명확하면 구독자 적어도 스폰서가 먼저 붙을 수 있음.

## 3. 채널 컨셉 정의

### 타겟
- K-pop/드라마가 아닌 **진짜 한국**에 관심 있는 외국인
- "한국은 왜 저래?" 라는 질문을 가진 글로벌 시청자

### 포지션 한 줄
> *"The only channel that explains WHY Korea is the way it is — using the actual textbooks Korean students study."*

### 기존 채널과의 차이

| 기존 채널 | 박씨 방송국 |
|-----------|------------|
| K-pop/드라마 → "한국 멋있다" | KR Merit + 교과 → "왜 저러는지" |
| 한국어 채널 → "이렇게 말해요" | 이유를 설명하는 스토리 |
| 문화 채널 → "한국은 이래요" (what) | 구조적 원인 설명 (why) |

**전부 "what"만 설명. 이 채널은 "why".**

## 4. 차별화 요소

### 콘텐츠 구조
- **소재:** 한국 고등학교 교과 (대학 아님 — 최소한의 지적 베이스라인)
- **스토리:** 교과 개념 → KR Merit 연결 → 한국 사회 작동 방식 설명
- **형식:** 웹툰 + 화면 녹화 + AI 활용 과정 공개

### 퍼스나 차별화

| 요소 | 임팩트 |
|------|--------|
| 79년생 (1979년생) 아저씨 | "나도 할 수 있겠다" — 중장년층 공감, 신뢰감 |
| 젊지 않음 | 아이돌/팬덤 문화와 완전히 다른 시각 |
| AI로 웹툰 직접 제작 | 말이 아니라 결과물로 증명 |
| 모듈/드라이버 시스템 보유 | 체계 있는 사람으로 인식 |
| AI 네이티브처럼 활용 | 투명한 과정 공개 = 요즘 트렌드 |

**전 세계에 거의 없는 조합:**
> 79년생 + AI 네이티브처럼 씀 + 결과물 있음 + 한국 문화 WHY 설명

## 5. 콘텐츠 구조

```
메인 영상 (10~15분)
  └ 교과 개념 하나 + KR Merit 연결 + AI 활용 과정 공개

쇼츠
  └ "한국 고교생은 이걸 외워야 함" 같은 자극적 클립

웹툰
  └ 같은 개념을 만화로 → Pinterest/Instagram 배포
```

### 웹툰 스토리 구조
- 스토리 안에 교과목 + 상식 + KR Merit이 녹아있음
- "한국이 왜 저 지랄을 하고 있는지" 이유가 스토리로 설명됨
- 시리즈 전체가 다큐멘터리 구조로 작동

### 콘텐츠 연결 예시
```
삼성 갑질 뉴스 → 교육 구조 → KR Merit 연결
수능 열풍      → 왜 저러는지 교과서로 설명
한국 직장 문화 → 고교 때 주입된 것들
```

## 8. 전략 요약

1. **검색형 콘텐츠**로 시작 — 알고리즘이 먼저 픽업
2. **쇼츠 병행** — 노출 확대
3. **웹툰 SNS 배포** — Pinterest/Instagram으로 외부 유입
4. **구독자보다 타겟 명확성** 우선 — 스폰서 먼저 붙을 수 있음
5. **AI 활용 과정 공개** — 채널의 메타 콘텐츠로도 작동
